# 通用快捷键

飞书文档包含4种类型，分别是：**文档**、**表格**、**多维表格**和**思维笔记**。当我们新建一个上面谈到的文件时，那么可以通过 `control+/` 的方式，来打开快捷键。在右侧面板中，将可以看到对应的各类型文件的快捷键操作。

更详细的使用方式可戳此链接：https://www.feishu.cn/hc/zh-CN/articles/360033771493
